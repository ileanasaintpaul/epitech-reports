## Exercices du Bootstrap 1

J'ai réalisé les exercices suivants du Bootstrap 1 :

- Affichage de la mesure de distance effectuée par un capteur ultrason sur le terminal série.
- Utilisation d'un potentiomètre pour contrôler le déplacement d'un signal lumineux sur une série de LEDs.
- Lecture des chiffres pressés sur un clavier (keypad) et affichage sur un écran LCD.
- Création d'un jeu Snake utilisant une matrice de LEDs pour l'affichage et un joystick pour les contrôles.
